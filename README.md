# CodeConductor MCP

CodeConductor MCP bundle for Claude Desktop. Install the `.mcpb` to auto-register the orchestrator and enable IDE-aware operations (file open/read/write/search and Pro-level git/command tools when licensed).

## Download
- Latest bundle: codeconductor-1.0.0.mcpb

## Install in Claude Desktop
1) Open Claude Desktop → Settings → Extensions.
2) Click “Install Extension”.
3) Pick `codeconductor-1.0.0.mcpb`.
